🧊 3D Tetris – консольный трёхмерный тетрис

**Классический тетрис, перенесённый в трёхмерное пространство** – на **7 языках программирования**.  
Играйте на поле 6×6×6, управляйте объёмными фигурами, заполняйте слои и устанавливайте рекорды!

---

## 🎮 Геймплей

- **Поле**: 6 (X) × 6 (Z) × 6 (Y) – ширина, глубина и высота.
- **Фигуры**: стандартные тетрамино (I, O, T, L, J, S, Z), адаптированные для 3D (толщина 1 слой).
- **Управление**:
  - `←` / `→` – движение по оси X (влево/вправо)
  - `W` / `S` – движение по оси Z (вперёд/назад)
  - `Пробел` – вращение фигуры вокруг вертикальной оси (Y)
  - `↓` – мгновенное падение вниз
  - `P` – пауза, `R` – рестарт, `Q` – выход
- **Цель** – заполнить горизонтальные слои (Y) полностью; заполненный слой исчезает, а очки начисляются.
- **Счёт** – количество уничтоженных слоёв. Рекорд сохраняется в файл.

## 🖥️ Отображение

Игровой экран показывает **вид сверху** на поле, где каждый квадрат обозначает столбец, а его цвет/символ показывает высоту (количество кубиков в этом столбце).  
Текущая фигура отображается в виде проекции на плоскость XZ с цветом.

## 🚀 Запуск

| Язык       | Файл          | Команда запуска                              | Зависимости                |
|------------|---------------|----------------------------------------------|----------------------------|
| Python     | `tetris3d.py` | `python tetris3d.py`                         | `keyboard`, `colorama`     |
| JavaScript | `tetris3d.js` | `node tetris3d.js`                           | `keypress`                 |
| Java       | `Tetris3D.java`| `javac Tetris3D.java && java Tetris3D`      | `jline`, `gson`            |
| C++        | `tetris3d.cpp`| `g++ -std=c++17 tetris3d.cpp -o tetris3d && ./tetris3d` | (Windows: `conio` / Unix: `ncurses`) |
| C#         | `tetris3d.cs` | `csc tetris3d.cs && tetris3d.exe`           | .NET SDK                   |
| Go         | `tetris3d.go` | `go run tetris3d.go`                         | `keyboard`                 |
| Rust       | `tetris3d.rs` | `cargo run`                                  | `crossterm`, `rand`, `serde_json` |

🛠️ Продвинутые функции
Полноценное 3D-поле – фигуры перемещаются по трём осям.

Вращение – фигура поворачивается вокруг вертикальной оси.

Удаление слоёв – заполненные горизонтальные слои исчезают.

Счёт и рекорд – сохраняются в JSON-файл.

Цветной вывод – каждый слой имеет свой оттенок (интенсивность).

Плавное падение – скорость увеличивается с ростом счёта.
